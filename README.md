## Running the Program

**IMPORTANT: Have at least java 1.8 when running**

### Windows:

1. Open command prompt
2. Move to the directory with the jar file
3. Run the command `java -jar Search.jar [-cost] <BFS|DFS|UCS|GS|A-star|HCS|SA> <inputfile>`

### MacOS:
1. Open Terminal
2. Move to the directory with the jar file
3. Run the command `java -jar Search.jar [-cost] <BFS|DFS|UCS|GS|A-star|HCS|SA> <inputfile>`
